## 4. Home23 Integration Points

### Code Added for Home23 (in evobrew source)

**server/server.js lines 76-90 — Home23 config loader:**
```
let home23Config = null;
const home23ConfigPath = path.join(__dirname, '..', 'config.json');
if (fsSync.existsSync(home23ConfigPath)) {
  home23Config = JSON.parse(fsSync.readFileSync(home23ConfigPath, 'utf8'));
```
Loads `config.json` (auto-generated by Home23) for brain paths and agent config. This is **graceful** — if the file doesn't exist or lacks `_generated`, it's a no-op.

**server/server.js lines 4369-4376 — Home23 brain dirs:**
Adds Home23 agent brain paths to BRAIN_DIRS allowlist from `home23Config.brain.agentBrains`.

**server/server.js lines 5369-5376 — Auto-load default brain:**
Auto-loads default brain from Home23 config if no brain is loaded yet.

**server/server.js lines 3684, 3698, 4414 — Agent brain info in API responses:**
Exposes `agentBrains` from home23Config in brain/status API endpoints.

### Base Evobrew Features (not Home23-specific)

**server/providers/adapters/local-agent.js (192 lines):**
Full LocalAgentAdapter for connecting to local agent processes via HTTP+SSE. This is base evobrew functionality — it works with any HTTP agent, not just Home23 agents.

**server/server.js lines 4908-4942, 4991-5127 — Agent discovery + setup routes:**
`/api/setup/scan-agents`, `/api/setup/local-agent/save`, `/api/setup/local-agent/test`, `/api/setup/local-agent/remove` — generic local agent setup, not Home23-specific.

**public/index.html line 14947 — ?agent= query param:**
Handles `?agent=` URL parameter for Home23 dashboard integration (agent selector pre-fill). Minimal, graceful.

### Assessment

The Home23 integration is clean and minimal. It's a ~30-line config loader that gracefully degrades when not running under Home23. No evobrew functionality breaks without it. The `local-agent` system is generic. **No changes needed for standalone operation.**

## 9. Standalone Functionality Assessment

Evobrew works standalone. The Home23 integration is purely additive:
- `config.json` loader gracefully degrades (no file = no-op)
- All core features (AI chat, brain loading, terminal, file ops) work without Home23
- Local agent system is generic HTTP/SSE, not Home23-coupled
- Setup wizard handles all provider configuration independently
- `~/.evobrew/config.json` is the primary config path when not under Home23

The `?agent=` URL parameter handling is the only Home23-specific UI code, and it's a harmless no-op without the parameter.
