🧠 Feed the Brain - Inject Documents

Upload files or folders to inject into the running brain's memory. Documents will be analyzed and integrated without restart.

Select Documents

Supports: Text (.txt, .md, .json, .py, .js, etc.), Binary (.pdf, .docx, .xlsx)
Select a folder to upload all its files recursively

Selected Files:

Specify how you want the documents analyzed, or leave empty for comprehensive analysis

How It Works

  1. Select individual files or an entire folder to inject
  2. Files are written to runtime/outputs/injected/
  3. Appropriate agent is spawned:
    • Text files (.txt, .md, .json, .py, etc.) → DocumentAnalysisAgent
    • Binary files (.pdf, .docx, .xlsx) → SpecializedBinaryAgent
    • Mixed files → Both agents sequentially
  4. Agent(s) read files, extract content, create memory nodes
  5. Brain integrates knowledge, coordinator may spawn follow-up agents
  6. No restart required - live injection into running system!
